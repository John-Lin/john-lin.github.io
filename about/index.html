Side Projects

Wall of Sheep

A packet sniffer to eavesdrop on unencrypted data in the packets and post on the web page. We make use of a powerful API provided by RethinkDB, Firebase to store and sync data in realtime. You can check out the demo page. Origilal written in Python but now rewrite in Node.js since performance issues.
